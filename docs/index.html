🏗 Architecture

Built for production

A layered architecture with resilience patterns, circuit breakers, and sandboxed execution.

Modular & Extensible

Every component is a first-class module. Swap providers, add skills, extend with plugins. The architecture handles failure gracefully at every layer.

  • Circuit breakers on all external calls
  • Exponential backoff + adaptive retry
  • Path traversal & injection protection
  • Privacy-first with local-only mode
  • Plugin system with sandboxed loading
  • Graceful degradation on provider failure

🔑 Configuration

Set up your API keys

Configure cloud LLM providers with API keys, or use Ollama / LM Studio for completely free local inference.

1 Environment Variables (Recommended)

.env  /  ~/.bashrc  /  ~/.zshrc
# Cloud Providers — pick one or more
OPENAI_API_KEY="sk-proj-..."
ANTHROPIC_API_KEY="sk-ant-api03-..."
OPENROUTER_API_KEY="sk-or-v1-..."
GOOGLE_API_KEY="AIza..."
XAI_API_KEY="xai-..."

# Optional: Set default provider
LINGUCLAW_PROVIDER="openai"

2 Configure via CLI

terminal
# Set your LLM provider
❯ linguclaw settings set llm.provider openai
# Set your API key
❯ linguclaw settings set llm.apiKey sk-proj-...

# Switch providers anytime
❯ linguclaw settings set llm.provider anthropic
❯ linguclaw settings set llm.apiKey sk-ant-api03-...

# View all settings
❯ linguclaw settings list
💡 Using Ollama or LM Studio? No API key needed — just install and run locally. LinguClaw auto-detects local providers on localhost:11434 (Ollama) or localhost:1234 (LM Studio).

3 Step-by-Step: Get Your API Key

🌐
OpenRouter (Recommended)
100+ models with a single key. Best for getting started.
1. Go to openrouter.ai/keys
2. Sign up or log in with Google/GitHub
3. Click "Create Key"
4. Copy the key (starts with sk-or-v1-)
OPENROUTER_API_KEY
🤖
OpenAI
GPT-4o, o1, GPT-5
1. Go to platform.openai.com/api-keys
2. Sign up and add billing
3. Click "Create new secret key"
4. Copy the key (starts with sk-proj-)
OPENAI_API_KEY
🧠
Anthropic
Claude 3.5 / Claude 4
1. Go to console.anthropic.com
2. Sign up and add billing
3. Navigate to Settings → API Keys
4. Copy the key (starts with sk-ant-)
ANTHROPIC_API_KEY
🔵
Google
Gemini 1.5 / 2.5
1. Go to aistudio.google.com/apikey
2. Sign in with Google account
3. Click "Create API Key"
4. Copy the key (starts with AIza)
GOOGLE_API_KEY
✖️
xAI
Grok models

3. Create an API key
4. Copy the key (starts with xai-)
XAI_API_KEY

Ollama
Free, 100% local, no cloud

4 Quick Setup Example (Full)

terminal — complete setup in 30 seconds
# Step 1: Install globally
❯ npm install -g linguclaw

# Step 2: Set provider (choose one)
❯ linguclaw settings set llm.provider openrouter

# Step 3: Set your API key
❯ linguclaw settings set llm.apiKey sk-or-v1-your-key-here

# Step 4: Verify everything works
❯ linguclaw status

# Step 5: Launch!
❯ linguclaw web
# Or run a task directly:
❯ linguclaw dev "refactor the auth module"
