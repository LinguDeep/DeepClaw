# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Multi-agent AI system with specialized agents (Architect, Coder, Reviewer, Tester)
- Git integration with blame, diff, and branch management
- Code refactoring engine with 15+ operations
- C# language support with LINQ and async analysis
- Semantic memory with TF-IDF search
- Task scheduler with cron and interval jobs

## [1.0.0] - 2024-01-15

### Added
- Initial release with TypeScript/JavaScript support
- Python, Rust, Go, Java, C++ language parsers
- Security vulnerability detection
- AST-based code analysis
- Web UI for code visualization
- Plugin system architecture
- SQLite-backed semantic memory
- LLM integration (OpenAI, Ollama)
- Multi-language code graph visualization

### Security
- SQL injection detection for Python
- XSS vulnerability detection
- Hardcoded secret detection
- Insecure dependency scanning

## [0.9.0] - 2024-01-01

### Added
- Beta release with core functionality
- Basic CLI interface
- Test suite with Jest
- CI/CD pipeline with GitHub Actions

### Fixed
- TypeScript compilation errors
- Memory initialization issues
- Plugin loading race conditions

## [0.1.0] - 2023-12-01

### Added
- Project initialization
- Basic project structure
- Core engine interfaces
- Proof of concept parser
