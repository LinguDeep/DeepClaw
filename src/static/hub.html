🔌 Easy Integration

Add a skill in 3 steps

Extend LinguClaw with your own logic — zero config, hot-reloadable.

1

Extend BaseSkill

Create a TypeScript class with NAME, DESCRIPTION, VERSION, and an execute() method. That's it.

2

Drop in plugins/

Place your file in the plugins directory or ~/.linguclaw/plugins/. Auto-discovered on startup.

3

Use Immediately

Your skill is available to all agents instantly. Hot-reload means no restart needed during dev.

Simple, powerful
skill API

Every skill follows the same pattern: define metadata, implement execute(), and declare a schema. LinguClaw handles the rest — discovery, validation, sandboxing.

  • TypeScript-first with full type safety
  • Automatic input validation from schema
  • Sandboxed execution with resource limits
  • Built-in versioning and dependency management
  • Hot-reload during development
web-scraper.ts
import { BaseSkill, SkillInput, SkillResult } from 'linguclaw';

export class WebScraperSkill extends BaseSkill {
  NAME = 'web-scraper';
  DESCRIPTION = 'Scrape websites with CSS selectors';
  VERSION = '2.1.0';
  AUTHOR = 'LinguClaw';
  TYPE = 'automation' as const;

  async execute(input: SkillInput): Promise<SkillResult> {
    const { url, selector } = input;
    const page = await this.browser().goto(url);
    const data = await page.$$eval(selector, els =>
      els.map(el => el.textContent)
    );
    return { success: true, data };
  }

  getSchema() {
    return {
      input: {
        url: { type: 'string', required: true },
        selector: { type: 'string', required: true }
      }
    };
  }
}
